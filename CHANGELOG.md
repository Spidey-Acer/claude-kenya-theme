# Changelog

All notable changes to the Claude Kenya theme are documented here.

## [1.0.1] — 2026-03-27

### Fixed
- Extension icon updated to CCK community logo
- Repository URLs corrected to match GitHub origin
- README version references updated

### Added
- `.vscodeignore` for leaner VSIX packaging
- `.gitignore` for clean version control
- Marketplace badges in README

## [1.0.0] — 2026-03-26

### Added
- **Savanna Dusk** — dark variant with 215 workbench colors, 59 token scopes, 19 semantic tokens
- **Savanna Dawn** — light variant with 170 workbench colors, 44 token scopes, 16 semantic tokens
- Full semantic highlighting support for LSP-enabled languages
- Optimized syntax highlighting for JavaScript, TypeScript, Python, Rust, Go, HTML, CSS, JSON, YAML, Markdown, Shell, Dockerfile
- Complete workbench theming: editor, sidebar, activity bar, tabs, terminal, status bar, panels, breadcrumbs, minimap, peek view, merge conflicts, debug toolbar
